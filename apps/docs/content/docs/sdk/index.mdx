# SDK

The `@recur/sdk` package provides everything you need to integrate Recur Protocol into your application.

## For subscribers

Build transactions that create on-chain subscriptions and manage token delegation:

- [Quickstart](/docs/sdk/quickstart) — install, configure, and create your first subscription
- [Subscribe](/docs/sdk/subscribe) — build subscribe transactions
- [Cancel](/docs/sdk/cancel) — cancel active subscriptions

## For merchants

REST API wrappers authenticated with API keys:

- [Plans](/docs/sdk/plans) — create and manage subscription plans
- [Subscriptions](/docs/sdk/subscriptions) — query subscriber data

## Webhooks

- [Webhooks](/docs/sdk/webhooks) — verify and handle payment event notifications
